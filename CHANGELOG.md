# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-01

### Added

- PowerShell bedtime enforcement agent
- Watchdog-based recovery flow
- Native WinForms desktop dashboard
- Weekday and weekend schedule support
- Warning, lock, shut down, and sign out enforcement modes
- Quick test mode with schedule restore
- Skip cooldown and schedule-change cooldown rules
- Autostart install, remove, and repair actions
- Tray support, log viewer, and shared config/state/status files
- GitHub-ready project docs, roadmap, and MIT license
