# SleepyNight v0.1.0

First public build of SleepyNight.

## Highlights

- native desktop dashboard
- bedtime enforcement agent and watchdog
- weekday and weekend schedules
- warning pipeline before restriction starts
- lock, shut down, and sign out enforcement modes
- autostart install / remove / repair actions
- quick test mode and live status view

## Notes

- designed for Windows
- autostart installation may trigger a UAC prompt
- the desktop app works together with the bundled PowerShell scripts and config files included in the release package
